# CZ（Changpeng Zhao）决策记录与行动

**研究日期**: 2026年4月14日

## 1. 关键决策（按时间顺序）

### 决策1: 全仓买入比特币（2013-2014年）
- **背景**: 2013年在扑克牌局上听Bobby Lee介绍比特币
- **行动**: 卖掉上海唯一房产，将所有资产换成BTC
- **当时价格**: 约$600/BTC
- **结果**: 资产增长数十倍
- **决策逻辑**: 极度看好加密货币未来，All-in式押注
- **事后反思**: CZ最经典的"高信念决策"
- **来源**: 一手（多次访谈、回忆录）

### 决策2: 创办币安（2017年7月）
- **背景**: 离开OKCoin，看到交易所市场空白
- **行动**: ICO筹集$1500万，180天成为全球最大交易所
- **决策逻辑**: "速度比完美更重要"，快速迭代
- **结果**: 成功，但最终因合规问题付出代价
- **来源**: 一手

### 决策3: 离开中国（2017年底）
- **背景**: 中国禁止加密货币交易
- **行动**: 举家迁离，辗转日本→马耳他→迪拜
- **决策逻辑**: "Build where you're welcome"
- **结果**: 在海外获得了自由发展空间
- **来源**: 一手

### 决策4: 投资Elon Musk收购Twitter（2022年）
- **背景**: Musk收购Twitter，CZ投资$5亿
- **决策逻辑**: 支持言论自由，看好社交平台+加密的协同
- **结果**: Twitter/X最终未深度整合加密（截至2026年）
- **来源**: 一手（CZ确认）

### 决策5: 抛售FTT引爆FTX危机（2022年11月）
- **背景**: 发现FTX/Alameda的"最近披露"
- **行动**: 公开宣布将出售FTT，引发FTX挤兑
- **决策逻辑**: 保护币安和用户利益，果断止损
- **结果**: FTX在数天内崩塌，SBF后来被判25年
- **事后反思**: "We won't pseudo-serve people"
- **争议**: 被质疑是否是"蓄意引爆"
- **来源**: 一手

### 决策6: 放弃收购FTX（2022年11月）
- **背景**: 宣布收购FTX一天后放弃
- **决策逻辑**: 尽调后发现FTX问题远超预期
- **结果**: 正确决定，避免了被FTX拖垮
- **来源**: 一手

### 决策7: 认罪并辞去CEO（2023年11月）
- **背景**: DOJ多年调查后提出认罪协议
- **行动**: 承认违反AML法，辞去Binance CEO，接受4个月监禁
- **决策逻辑**: "承担责任，结束不确定性"
- **结果**: 币安存活，个人失去自由但保留了财富和影响力
- **争议**: 律师称同类指控从未导致入狱，CZ是否被"过度惩罚"？
- **来源**: 一手

### 决策8: 主动赴美服刑
- **背景**: 认罪后本可留在阿联酋（无引渡协议）
- **行动**: 主动飞往美国接受判决
- **决策逻辑**: "我做了错事，我要面对后果"
- **结果**: 赢得了公众尊重，为赦免铺路
- **来源**: 一手

### 决策9: 出狱后启动Giggle Academy（2024年）
- **背景**: 出狱后没有回归Binance
- **行动**: 创办免费教育平台Giggle Academy
- **决策逻辑**: 回归"建设"初心，用教育赋能
- **结果**: 仍在发展中
- **来源**: 一手

### 决策10: 创建YZi Labs（2025年）
- **背景**: 用个人资金进行加密/科技投资
- **行动**: 2025年投资约70个项目，评估1000+项目
- **决策逻辑**: 作为"家族办公室"式投资，不对外开放
- **争议**: FT报道可能对外开放，CZ否认
- **来源**: 一手

## 2. 决策模式分析

### 2.1 速度优先型决策
- 180天建成全球最大交易所
- FTX事件中24小时内做出抛售→放弃收购的决定
- "速度比完美更重要"是核心原则

### 2.2 高信念型押注
- 卖房买BTC: 全仓押注
- 创办币安: 全身心投入
- 拒绝回国（中国加密禁令后）: 坚持原则

### 2.3 务实转向型决策
- 从"不做期货"→做期货
- 从"不做法币"→接入法币
- 从"去中心化"→接受合规
- 从"Binance CEO"→教育家和顾问
- 转向逻辑: 用户需求/市场现实 > 固定立场

### 2.4 承担责任型决策
- 认罪: 不归咎他人
- 赴美服刑: 本可逃避
- 辞职: 将公司存续放在个人利益之上
